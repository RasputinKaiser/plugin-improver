# plugin-tune-triggers

Source: [`../../skills/plugin-tune-triggers/SKILL.md`](../../skills/plugin-tune-triggers/SKILL.md)

## Purpose

Tunes skill descriptions and trigger behavior in an existing plugin so each skill fires
exactly when it should, backed by a should/shouldn't-trigger test matrix.

## When it fires

When a skill triggers too often, never triggers, or overlaps with another skill, or when
asked to improve descriptions, trigger phrases, or implicit invocation. Not for rewriting a
skill's body instructions.
